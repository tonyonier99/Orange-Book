# 🍊 小橘書防災清單 (Orange Book Checklist)

> **這是一個為個人與家庭設計的「可重複使用」防災物資檢查工具。**
> 靈感源自國防部「全民國防應變手冊」，並將其轉化為數位化、互動式的操作體驗。

## 🚀 成果展示 (Outcome Display)
*   **網頁版連結：** [https://orange-book-checklist.zeabur.app/](https://orange-book-checklist.zeabur.app/)
*   **手排版優化：** 針對手機瀏覽進行深度優化，支援流暢的移動端勾選體驗。

## ✨ 功能說明 (Feature Description)
1.  **雙重模式切換**：提供「居家物資儲備」與「緊急避難包」兩種模式，各別紀錄進度。
2.  **進度追蹤系統**：頂部即時顯示準備進度百分比，手機版進度條自動固定於螢幕上方。
3.  **自動提醒機制**：可下載 `.ics` 檔案或一鍵加入 Google 日曆，設定 6 個月後的定期檢查提醒。
4.  **下載與列印**：支援將勾選成果產生成高品質圖片下載，或直接列印為紙本清單。
5.  **自定義項目**：使用者可自由新增個人化所需物資。
6.  **一鍵分享**：透過系統原生分享 API，輕鬆將工具推薦給家人好友。

## 🛠️ 開發工具與架構
*   **核心技術**：HTML5, CSS3 (Vanilla CSS), JavaScript (Vanilla JS)
*   **關鍵套件**：`html2canvas` (圖片生成)
*   **輔助開發**：使用 **Google DeepMind - Antigravity (Gemini)** AI 配對編程開發。
*   **部署平台**：Zeabur (自動 CI/CD)

## 🎨 設計考量
*   **配色方案**：採用鮮明的橙色與黑色，象徵「警示與準備」，同時維持「小橘書」的品牌美學。
*   **高可用性**：考慮到災害發生時的各種情境，設計了離線下載圖片與紙本列印功能。
